# CLI Reference

## run

```bash
dftflow run input.xyz --config run_config.yaml
```

Key options:

- `--background`: enqueue a background run
- `--profile`: record SCF/gradient/Hessian timing
- `--resume <RUN_DIR>`: resume an existing run
- `--force-resume`: allow resume from completed/failed states
- `--queue-priority <N>`: queue priority
- `--queue-max-runtime <SEC>`: queue timeout
- `--scan-dimension ...`, `--scan-mode ...`, `--scan-grid ...`: CLI scan overrides

## status

```bash
dftflow status

dftflow status --recent 5
```

## queue

```bash
dftflow queue status
dftflow queue cancel <RUN_ID>
dftflow queue retry <RUN_ID>
dftflow queue requeue-failed
dftflow queue prune --keep-days 30
dftflow queue archive
```

## validate-config

```bash
dftflow validate-config run_config.yaml
```

## doctor

```bash
dftflow doctor
```

## smoke-test

```bash
dftflow smoke-test --smoke-mode quick
```

## scan-point (advanced)

Use a manifest generated by `scan.executor: manifest`.

```bash
dftflow scan-point --manifest path/to/scan_manifest.json --index 0
```
